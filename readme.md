
# Spring Study Platform
![Logo](https://imj31us4am50.github.io/imgs/logospringstudy.png)

Study platform designed to make learning more organized and efficient, featuring tools such as a Pomodoro timer and access to study materials. Created as a competition project, the platform showcases the concept and core features aimed at supporting students during exam preparation.


## Team

- [@imj31us4am50](https://github.com/imj31us4am50) - front-end, layout & functionality
- [@shev0k](https://github.com/shev0k) - UI/UX & prototyping
- [@AvramAlexandruV](https://github.com/AvramAlexandruV) - back-end & server setup

## Features

- Personalized study plan creation
- Pomodoro timer and flexible study timer
- Structured study resources (Grades IX–XII)
- Simple, accessible navigation structure
- Responsive design for all devices
- Offline usability via browser cache support
- Feedback system for user suggestions
